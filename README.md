# AspNetCore-ECommerce 🛒

ASP.NET Core MVC ve Entity Framework Core kullanılarak geliştirilmiş, modern, ölçeklenebilir ve katmanlı mimariye uygun bir e-ticaret web uygulamasıdır.

Bu proje, yalnızca temel CRUD işlemlerinden ibaret olmayıp; gerçek dünya senaryolarına yakın, admin paneli, kullanıcı deneyimi ve harici servis entegrasyonlarıyla zenginleştirilmiştir.

# Projenin Amacı 📌

Bu proje ile hedeflenen:

Gerçek bir e-ticaret sisteminin temel yapı taşlarını öğrenmek

Katmanlı mimari (N-Tier Architecture) mantığını uygulamak

Backend + Frontend + Database + API entegrasyonlarını bir arada kullanmak

Dinamik, sürdürülebilir ve genişletilebilir bir sistem geliştirmek

 # Proje Mimarisi 🏗️

Proje, sürdürülebilir ve okunabilir olması için katmanlı yapı prensibine uygun geliştirilmiştir:

Controllers → İş mantığı ve routing

Models → Entity ve ViewModel yapıları

Views → Razor tabanlı kullanıcı arayüzü

Data → DbContext ve veritabanı işlemleri

wwwroot → Statik dosyalar (CSS, JS, görseller)
